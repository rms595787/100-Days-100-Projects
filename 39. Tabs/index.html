HTML

HTML (HyperText Markup Language) is the standard language for creating and structuring web pages. It uses a system of tags to define different elements, such as headings, paragraphs, and images, that a web browser uses to display content. HTML is the foundation of every website, providing the basic structure for content on the web.

CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation and formatting of content written in a markup language, most commonly HTML. It is a foundational technology of the World Wide Web, alongside HTML and JavaScript.

JAVASCRIPT

JS is the common abbreviation for JavaScript, a versatile, high-level programming language and one of the core technologies of the World Wide Web, alongside HTML and CSS. It is primarily known for making web pages dynamic and interactive.

REACT

React is an open-source JavaScript library for building user interfaces (UIs), developed and maintained by Meta (formerly Facebook) and a large community of developers. It is used for creating dynamic, interactive, and high-performance user interfaces for web and mobile applications.

NODE

"Node" is a versatile term with different meanings depending on the context, most commonly referring to a connection point in a network or a data structure, or, when capitalized as Node.js, a popular JavaScript runtime environment.
